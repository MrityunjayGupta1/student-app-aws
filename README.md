# Student Management App on AWS

Simple student management app deployed on AWS EC2.

## AWS Services Used
- EC2 - runs the Flask app
- Security Groups - controls traffic
- CloudWatch - monitors the server

## How to Run
pip install flask
python app.py

## Project Structure
app.py - main Flask application
requirements.txt - python dependencies
